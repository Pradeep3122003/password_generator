## üß∞ Getting Started

### Requirements

* Python 3.x installed (for `password.py`).
* Bash shell (or compatible) for the `password` script.
* (Optional) `chmod +x password` to make shell script executable.

### Installation

```bash
# Clone the repo
git clone https://github.com/Pradeep3122003/password_generator.git
cd password_generator
```

## ‚úèÔ∏è Usage

### Python version

```bash
# Generate a password using default settings
python3 password.py

# Generate a password using custom length (example: 15)
python3 password.py 15
```

### Shell version

```bash
# Make script executable (once only)
chmod +x password

# Run the script (default length)
./password

# Run with custom length (example: 20 chars)
./password 20
```

### C++ version

```bash
# Compile the code
g++ password.cpp -o pass

# Generate a password using default settings
./pass

# Generate a password using custom length (example: 15)
./pass 15
```
